# Project Hampden

*A museum-quality digital reconstruction of the Handley Page Hampden Mk I.*

---

## Overview

Project Hampden is an open-source effort to create the most accurate publicly available digital reconstruction of the Handley Page Hampden Mk I medium bomber as it appeared during the opening months of the Second World War.

This project goes beyond creating a 3D model. The goal is to document and digitally restore the aircraft using historical references, engineering drawings, photographs, and surviving technical documentation.

The finished project will include:

- Blender source files
- Engineering blueprints
- PBR texture sets
- Historical documentation
- Python tools for Blender
- Export formats for visualization, simulation, and 3D printing

---

## Why This Project?

The Handley Page Hampden is one of the least represented RAF bombers of World War II.

While several digital models exist, few are built from verified historical references, and none are intended as a comprehensive engineering reconstruction.

Our goal is to create a model that is useful for:

- Aviation historians
- Scale model builders
- Museums
- Flight simulation developers
- Blender artists
- WWII enthusiasts
- Students of aircraft design

---

## Project Philosophy

Every major modeling decision should be supported by historical evidence whenever possible.

Where references conflict or are incomplete, assumptions will be documented.

Historical accuracy is more important than speed.
